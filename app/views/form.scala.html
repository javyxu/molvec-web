@(api: String)
Batch API 
curl --data-binary @@{image} -H Content-Type:image/png @api/{engine}
where {image} is the input image file and {engine} is one of the supported values: molvec, osra, imago, or all. Note that the value for Content-Type can be any supported image formats; it doesn't have to match the actual mime-type of the specified image.
